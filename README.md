# Fórum 

## Aplicação 
<b>Acesso:</b> http://localhost:8080/#/topic
* Criar tópico de discussão 
* Responder a tópico
* Listar tópicos

## Rest API 
<b>Acesso:</b> http://localhost:8080/api/topics/
* <b>Criar tópico/resposta</b>
  - POST    para    api/topics/
* <b>Listar tópicos</b>
  - GET    para     api/topics/
* <b>Consultar tópico</b>
  - GET    para     api/topics/[id do tópico]
